# 🎹 Tonik

**The keyboard that's always in key.** Every key on Tonik is a note of the scale you choose, so nothing you play can sound harsh or wrong — all in a single HTML file with no installs and no dependencies, just the Web Audio API.

**Live demo:** https://aakash7192.github.io/digiano/

## Features

- **Scale-aware keyboard** — pick a root note and scale (Major, Minor, Dorian, Lydian, and more); the three-row keyboard re-tunes itself so every key is in-scale
- **Chord pads** — number keys `1`–`8` play the scale's diatonic triads, labeled with their quality (m, °, +)
- **Two instruments** — Piano and Harmonium, synthesized live with oscillators and filters
- **Octave shift** — `Left Shift` / `Right Shift` (or the ◀ ▶ buttons) move all three rows lower or higher
- **Play with your computer keyboard** — `Q`–`I` high octave, `A`–`K` middle, `Z`–`,` low — or click/touch; on phones the keys reflow into large touch targets

## Studio

- **Four finishes** — Walnut, Ebony, Ivory, Indigo lacquer themes; your pick is remembered
- **Multitrack recorder** — record takes as note events; overdub while hearing your other tracks
- **Mix screen** — play/pause/resume with a playhead, per-track on/off switches, solo, faders, master fader
- **Download your piece** — the whole mix renders offline to a 16-bit WAV (`Space` plays/pauses)

## Run locally

Just open `index.html` in any browser.
